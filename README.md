# Discord Injection Builder

Builds an injection file that reports Discord account activity to a webhook.

## Usage

Provide a webhook URL and configure the output:

- `01` Python File
- `02` Executable File

Then choose the console mode:

- `01` With Console
- `02` No Console

Optionally add a custom icon for the executable.

## Requirements

- A valid Discord webhook URL
- Pillow (`pip install Pillow`)
- PyInstaller (`pip install pyinstaller`)

## Notes

- This plugin is intended for educational and authorized security testing only.
- Deploying injection files without consent is illegal.
